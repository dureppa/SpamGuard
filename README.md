# SpamGuard

Реализация наивного байесовского классификатора для фильтрации спама с нуля. Проект сфокусирован на понимании вероятностной модели, корректной предобработке текста и сравнении собственной реализации с эталонным решением.

## Цель проекта

Данный проект демонстрирует полную реализацию мультиномиального наивного байесовского классификатора без использования готовых высокоуровневых решений (например, `sklearn.naive_bayes.MultinomialNB`), за исключением этапа сравнения. Основные задачи:

- Реализовать конвейер предобработки текста: удаление пунктуации и чисел, фильтрация стоп-слов и лемматизация.
- Построить словарь на основе обучающей выборки с подсчётом частот слов.
- Вычислить априорные вероятности классов и условные вероятности слов с применением сглаживания Лапласа.
- Отфильтровать малозначимые слова по порогу частоты.
- Оценить качество собственной реализации и сравнить с реализацией из scikit-learn.

Задача фильтрации спама — классический кейс в машинном обучении, часто встречающийся на технических собеседованиях. Проект служит как учебное упражнение, так и демонстрация понимания основ ML.
